# 🌦️ Weather App

A simple and responsive Weather Application built using **HTML**, **CSS**, **JavaScript**, **Bootstrap**, and the **OpenWeather API**. It allows users to search for any city and view real-time weather information.

## 🚀 Features

* 🔍 Search weather by city name
* 🌡️ Displays current temperature
* 💧 Shows humidity
* 🌬️ Displays wind speed
* ☁️ Weather condition with icons
* 📱 Responsive design for mobile and desktop
* ⚡ Fast and user-friendly interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Bootstrap 5
* OpenWeather API
